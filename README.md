# Data Cleaning Using Python

## Project Overview
This project demonstrates the complete data cleaning process using Python and Pandas.

## Dataset Issues
- Missing Values
- Duplicate Records
- Inconsistent Gender Values
- Mixed Age Formats
- Inconsistent City Names
- Inconsistent Country Names
- Incorrect Data Types

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Data Cleaning Steps
1. Load Dataset
2. Check Missing Values
3. Remove Duplicates
4. Clean Gender Column
5. Clean Age Column
6. Clean City Column
7. Clean Country Column
8. Convert Date Columns
9. Handle Missing Values
10. Save Cleaned Dataset

## Output
Cleaned dataset with:
- No Missing Values
- Standardized Data
- Correct Data Types
